Step 0: If you have not set up the database, please do so first under src/database in the other github directory
Step 1: Download all the files and move them into htdocs\FYP\    (If don't have FYP folder, then just create one)
Step 2: Ensure your sql and apache is turned on in your XAMPP control (refer to src/database readme.txt)
Step 3: In your browser, type in localhost/fyp/main.php

You should be set to go.
